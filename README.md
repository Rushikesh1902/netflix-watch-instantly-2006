## Overview

**Netflix Watch Instantly (Beta)** is an alternate-history product prototype that explores a simple question:

> **What if the modern Netflix experience had launched in 2006?**

Rather than creating a retro skin, this project reimagines Netflix's core streaming experience within the technological and design constraints of the Windows XP era.

The result is a historically inspired, fully interactive streaming platform featuring an authentic 2006 interface, simulated broadband streaming workflow, and modern AI enhancements powered by Google Gemini.

---

## Features

- Authentic Windows XP-inspired user interface
- Fully functional streaming experience
- Windows Media Player-inspired video player
- Streaming initialization & buffering simulation
- CineMatch recommendation engine
- Movie browsing & filtering
- Search by title, genre, actor & year
- Personalized movie queue
- Continue Watching
- AI-powered Help Center
- Natural language movie search
- Recommendation explanations using Gemini
- Persistent user settings
- Responsive desktop-first experience

## Getting Started

Clone the repository

```bash
git clone https://github.com/Akash-256/netflix-watch-instantly-2006.git
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
VITE_GEMINI_API_KEY=YOUR_API_KEY
```

Start the development server

```bash
npm run dev
```

## Design Philosophy

This project was intentionally designed around the user experience of **late 2006**.

Research references included:

- Netflix (2006–2007)
- Windows XP
- Windows Media Player 10
- Windows Media Center
- Internet Explorer 7
- Early broadband streaming workflows

Every interface, interaction, and technical limitation was designed to reflect the capabilities of that period while preserving Netflix's modern product vision.

---

## AI Integration

Google Gemini is integrated as a modern enhancement layer for the prototype.

Current AI capabilities include:

- Natural language search
- Movie summaries
- Recommendation explanations
- Interactive Help Center

The core product experience remains historically authentic while AI improves usability for demonstration purposes.
